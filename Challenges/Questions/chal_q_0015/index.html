Consider the following three functions in C:

int f(void) {
    printf("f was called\n");
    return 1;
}

int g(void) {
    printf("g was called\n");
    return 2;
}


int h(void) {
    printf("h was called\n");
    return 3;
}

What will be printed to stdout when the following line is executed?

int a = (f() + g()) + h();


f was called
g was called
h was called


g was called
f was called
h was called

Either of the above two is possible
The three sentences can appear in any of the 6 possible orders
